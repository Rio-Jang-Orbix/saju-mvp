## 🎯 런칭 전 마지막 체크사항

### 1. **환경변수 설정**
```bash
# .env.local 파일 생성
NEXT_PUBLIC_APP_MODE=test
NEXT_PUBLIC_APP_URL=https://test.luvo.ai
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX  # 애널리틱스 추가시
```

### 2. **빌드 및 배포 테스트**
```bash
# 최종 빌드 확인
NEXT_PUBLIC_APP_MODE=test npm run build

# 프로덕션 서버 실행 테스트
npm start
```
